Caras Landing Page
  A sleek and high-performance magazine landing page inspired by the "Caras" lifestyle brand. This project serves as a robust prototype for a
  media-focused web application, featuring satirical news content and a refined editorial aesthetic.

  [![Netlify Status](https://api.netlify.com/api/v1/badges/780f8edf-3e15-4e7e-b020-116ee68203d7/deploy-status)](https://caras-landing-ts.netlify.app/)
  eploys)

  🚀 Key Features
   - Modern Tech Stack: Built with React 19 and TypeScript for maximum reliability and performance.
   - Polished UI/UX: Custom-styled using Styled Components with a focus on editorial typography (Barlow & Playfair Display).
   - Satirical Content: Features a dynamic news section with humorous takes on current events and entertainment.
   - Fully Responsive: Optimized for seamless viewing on all device sizes.
   - CI/CD Ready: Automatically deployed and hosted on Netlify.

  ♿ Accessibility & Standards (A11y)
  This project follows WCAG 2.1 AA guidelines to ensure a high-quality experience for all users:
   - Semantic HTML: Proper use of <main>, <nav>, <section>, and <article> landmarks for meaningful document structure.
   - Interactive Controls: Custom video players and search features implemented with semantic <button> elements and dynamic aria-labels.
   - Form Accessibility: Newsletter inputs use aria-describedby and aria-invalid to programmatically connect real-time validation prompts to the
     user.
   - Screen Reader Optimization: Used the "Redundant Link" pattern for headlines and implemented aria-live regions for status updates/alerts.
   - Color Contrast: Typography colors optimized to ensure a minimum contrast ratio of 4.5:1 (Normal Text) and 3:1 (Large/Bold Text).

  🛠️ Built With
   - React 19 - UI Framework
   - TypeScript - Type Safety

   - React Testing Library - Component Validation
   - Netlify - Hosting & Continuous Deployment
